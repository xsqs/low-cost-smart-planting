# low-cost-smart-planting
This is an open-source, home-based self-sufficiency system for jardinage​ focused on low-cost living. By growing fruits, vegetables, and other crops at home, it enables food self-sufficiency and helps relieve psychological stress.

# 开源智慧盆栽系统

## 项目简介
这是一个**低成本、可定制**的家庭智能盆栽解决方案，旨在帮助植物爱好者轻松实现家庭种植。

## 快速开始
1. **准备材料**
   - Arduino 开发板
   - 土壤湿度传感器
   - 小型水泵
2. **组装步骤**
   - 连接传感器
   - 上传代码
   - 配置浇水阈值
